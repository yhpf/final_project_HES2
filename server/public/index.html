Test that the API for the Todo app is working

This HTML/JS page will test the Todo API by listing all; then creating, finding, modifying, and deleting a todo attachment resource using the REST API.

The Todo application item has a Title, a Description, an Attachment, and a Deadline date. This test will be focused around the todo attachment file and the title, description, and deadline values will be added in the apitest.js file.

Please follow these steps to test:

  1. Make sure the Todo application is up and running. Can be done by opening the application in your browser.
  2. Make sure this test page is open via a file extension.
  3. Open the Inspector and then choose to view the Console
  4. See the Console for output as the process completes. (The process should complete without errors.)
